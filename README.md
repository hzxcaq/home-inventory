# Home Inventory System

家庭物品保存位置记录系统

## 项目结构

- `backend/` - Java Spring Boot 后端
- `mobile/` - React Native 移动应用
- `web/` - React Web 应用

## 功能特性

- 多地址/多房子管理
- 房间和储物位置管理（支持户型图编辑）
- 物品分类和管理
- 拍照记录物品
- 搜索和过滤
- 离线存储支持

## 技术栈

- Backend: Java Spring Boot
- Mobile: React Native
- Web: React
- Database: SQLite + Optional Cloud Sync
