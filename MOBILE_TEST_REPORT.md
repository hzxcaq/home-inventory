## 📱 **如何测试移动端**

### **方法1: Web预览 (推荐)**
1. 打开浏览器访问: `mobile-preview.html`
2. 查看移动端UI设计和功能列表
3. 模拟移动端交互体验

### **方法2: Android模拟器 (需要配置)**
```bash
# 需要先安装Android Studio和配置环境
cd mobile
npm run android
```

### **方法3: 物理设备**
```bash
# 使用Expo Go或React Native调试工具
cd mobile
npm start
# 扫描二维码连接设备
```
